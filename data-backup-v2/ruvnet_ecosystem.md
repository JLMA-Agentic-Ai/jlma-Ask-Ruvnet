# RuvNet Ecosystem Overview

The RuvNet ecosystem (https://github.com/ruvnet) contains over 160 repositories focused on Agentic AI, Robotics, and High-Performance Computing.

## Key Projects

### 1. ROS3 (Robot Operating System 3)
**Concept:** A ground-up rewrite of ROS2 in Rust.
**Goal:** To provide a memory-safe, high-performance, and real-time operating system for next-generation robotics.
**Key Features:**
-   Written in Rust for safety and concurrency.
-   Designed for autonomous swarms and edge robotics.

### 2. Agentic Synth
**Concept:** High-performance synthetic data generator.
**Goal:** To generate massive datasets for training AI models where real-world data is scarce.
**Key Features:**
-   Procedural generation of scenarios.
-   Physics-based simulation integration.

### 3. Tensor Compress
**Concept:** Quantum-inspired Tensor Train (TT) decomposition library.
**Goal:** To compress large neural networks for deployment on edge devices.
**Key Features:**
-   Implemented in Rust.
-   Reduces model size while maintaining accuracy.

### 4. .clinerules (Agentic Coding)
**Concept:** A structured guide for AI-human collaborative coding.
**Principles:**
-   **Simplicity:** Keep code clean and modular.
-   **Iteration:** Build in small, testable steps.
-   **Context:** Provide AI with clear context (like this file!).

### 5. Jujutsu (VCS)
**Concept:** Multi-Agent Collaborative Coding workflow.
**Goal:** To improve upon Git for complex, distributed AI development teams.
**Comparison:** Offers better conflict resolution and history rewriting than standard Git worktrees.
